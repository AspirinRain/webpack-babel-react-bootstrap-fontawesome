## Features

- Webpack 2
- Babel
- jQuery
- React
- Bootstrap
- Font Awesome

## Installation

``` bash
$ npm install
```

## Quick Start

**Build**

``` bash
$ npm run build
```

**Run**

``` bash
$ npm run start
```
